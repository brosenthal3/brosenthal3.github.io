About Me

My name is Ben Rosenthal and I am a front-end web developer. I always try to document my code with comments, so that it would be readable and understandable. All of my projects on this site are fully responsive for computers, tablets and phones, because I think it's very important for every website.

I'm a self taught developer and I've learnt everything I know from the internet. Some of the resources I used were FreeCodeCamp, The Odin Project and the Traversy Media YouTube channel.
